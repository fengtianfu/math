3 直线方程的各种形式

直线方程的一般形式是：

\[Ax + By + C = 0\]

从某种意义上，加上斜率

\[m = \frac{y_2 - y_1}{x_2 - x_1} = \tan\varphi\]

还有两点间的距离公式：

\[d(p_i, p_j) = \sqrt{(x_i - x_j)^2 + (y_i - y_j)^2}\]

这些就是我们所有需要知道的，但是从这些出发，推理出来的一些直线的形式，在表达上经常出现，值得我们去探讨．

3.1 两点形式

两点确定一条直线，也就确定了一个直线方程．

设：直线\(l\)上面两个点，\(p_1(x_1, y_1), p_2(x_2, y_2)\)

通过观察我们可以构造一个直线方程：

Theorem 8 (点点形式) 如果给出直线方程经过两点\((x_1, y_1), (x_2, y_2)\) 那么： \[y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)\]

代入\((x_1, y_1), (x_2, y_2)\)，等式成立．

我们可以看到，这个构造用到了斜率\(m = \frac{y_2 - y_1}{x_2 - x_1}\), 点\(p_1(x_1, y_1)\)，用斜率与另一个点也可以构造出直线：

\[y - y_2 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_2)\]

本文会从不同的角度去看一个问题，这种方式很重要，一是加深了问题的认识，第二是，通过这种锻炼，以后碰到问题，你可以从不同角度去分析，也许就找到了出路．

可以从三个角度证明这两个直线方程是相同的

前面我们证明过直线和直线方程是一一对应的关系，所以这两个直线方程必定是同样的．

从坐标系上看，如果有一个点，在加上和x轴的夹角，只能作出一条直线.

如果我们转换直线方程的形式，我们可以期望通过转换，我们会得到同样的一个方程．

\[y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)\]

两边同时加上\(y_1 -y_2\)

\[y - y_2 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1) + y_1 - y_2\] \[y - y_2 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1) + \frac{y_1 - y_2}{x_2 - x_1}(x_2 - x_1)\] \[y - y_2 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1) + \frac{y_2 - y_1}{x_2 - x_1}(x_1 - x_2)\] \[y - y_2 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_2)\]

这就证明了，这两个等式其实就是一个直线方程．

上面的直线方程在\(x_1 = x_2\)的时候，是无法使用的．我们可以转换：

\[y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)\]

两边同时乘以\(x_2 - x_1\)

\[(y - y_1)(x_2 - x_1) = (y_2 - y_1)(x - x_1)\]

若：\(x_1 = x_2\)

则：直线方程为\(x = x_1\)

3.2 斜率—点形式

Theorem 9 (斜率－点形式) 如果有直线方程经过一点\((x_1, y_1)\)，直线方程的斜率为m 那么直线方程为：\(y - y_1 = m(x - x_1)\)

我们可以从 两点形式 中推出 斜率－点 形式 \[y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)\]

\[m = \frac{y_2 - y_1}{x_2 - x_1}\]

\[y - y_1 = m(x - x_1)\]

3.3 斜率－截距形式

Theorem 10 (斜率－截距) 如果有直线方程经过y轴上一点\((0, b)\)，直线方程的斜率为m 那么直线方程为：\(y = mx + b\)

可以从 斜率点形式 ，来推出 斜率－截距 形式 \[y - y_1 = m(x - x_1)\]

\[y = m(x - x_1) + y_1 - mx_1\]

令\(b = y_1 -mx_1\)，则：

\[y = mx + b\]

当\(x = 0\)的时候，代入方程有\(y = b\)，那么直线上有一点为\((0, b)\)

3.4 交点形式

Theorem 11 (交点形式) 如果直线方程上面有两点，\((a, 0), (0, b)\) 那么 直线方程为：\(\frac{x}{a} + \frac{y}{b} = 1\)

再次，我们从不同角度来看这个问题．

设：直线 \(l\)

\(Ax + By + C = 0\)

\[\frac{x}{\frac{1}{A}} + \frac{y}{\frac{1}{B}} + C = 0\]

\[\frac{x}{\frac{C}{A}} + \frac{y}{\frac{C}{B}} + 1 = 0\]

\[\frac{x}{- \frac{C}{A}} + \frac{y}{- \frac{C}{B}} = 1\]

令\(a = - \frac{C}{A}, b = - \frac{C}{B}\), 则：

\[\frac{x}{a} + \frac{y}{b} = 1\]

我们可以从结果出发，来看这个问题．

\[\frac{x}{a} + \frac{y}{b} = 1\]

代入\((a, 0), (0, b)\), 满足方程．

这两个点一个在x轴上，一个在y轴上．

3.5 解析几何与欧几里得几何

解析几何可以用来证明欧几里得几何，绝大部分时候，我们在不同的地方使用它们．

若：有直线上两点\(A(x_1, y_1), B(x_2, y_2)\)，求AB的中点M坐标．

设该直线为\(Cx + Dy + F = 0\), M点的坐标为\(M(x_m, y_m)\)

设\(x_2 > x_1\)

\[AM = \sqrt{1 + (\frac{C}{D})^2}|x_m - x_1|\]

\(BM = \sqrt{1 + (\frac{C}{D})^2}|x_2 - x_m|\)

M是AB的中点

AM = BM

\(x_2 - x_m = x_m - x_1\)

\(x_m = \frac{x_1 + x_2}{2}\)

\(C(x_1 + x_2) + D(y_1 + y_2) + 2F = 0\)

\(C\frac{x_1 + x_2}{2} + D\frac{y_1 + y_2}{2} + F = 0\)

因此，\(x_m = \frac{x_1 + x_2}{2}, y_m = \frac{y_1 + y_2}{2}\)
